## 风险与依赖

| 风险/依赖 | 说明 | 应对措施 |
|-----------|------|----------|
| 视频文件存储 | 视频较大，需考虑存储空间 | 先用服务器本地存储，后期可迁移云存储 |
| 浏览器兼容性 | Web Speech API 兼容性 | 准备有道词典作为备选方案 |
| 学习码安全 | 8位数字可能被猜到 | 后续可增加密码或限制尝试次数 |
